## Logo 网站logo

```js
import { Logo } from 'sui-components';
```
### 何时使用

- 使用[sui-icons](https://git.code.oa.com/comment_frontend/sui-icons)图标库为网站添加logo时
- 需要自定义logo时, 欢迎给[sui-icons](https://git.code.oa.com/comment_frontend/sui-icons)提PR

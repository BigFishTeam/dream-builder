## Menu 菜单

```js
import { BrowserHistoryMenu } from 'sui-components';
```

### 何时使用

- 网站需要左侧菜单时使用
- 支持带二级标签（如：左侧菜单里的"基础"、"布局"、"导航"）和不带二级标签两种形式
- 菜单级别不能超过三级

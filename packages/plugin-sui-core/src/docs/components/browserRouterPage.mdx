## RouterPage 路由页面

```js
import { BrowserRouterPage } from 'sui-components';
import pageRoutes from 'routerPage.config.js';
```
### 何时使用

- 需要根据路由配置渲染页面时
- 默认根据路由配置文件routerPage.config.js渲染全部页面，同时支持业务方自行渲染动态子页面

### 使用方法

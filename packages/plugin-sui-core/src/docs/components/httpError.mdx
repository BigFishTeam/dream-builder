## HttpError 错误页

```js
import { HttpError } from 'sui-components';
```
### 何时使用

- 当页面请求出错时，根据http返回码渲染错误页面

### 使用方法

<CodeBox
  title="http错误"
  demo={[
    {
      component: <HttpError errorCode="500" />,
      code: `import { HttpError } from 'sui-components';
ReactDOM.render(<HttpError errorCode="500"/>, document.getElementById('app'));`
    }
  ]}
  api={[
    {
      title: '',
      data: [
        {
          key: '1',

    }
  ]}

### 如何使用使用

- SUI 内置一套 echarts 主题色，推荐使用<a href="https://echarts.baidu.com/" target="_blank" >echarts</a>。
- 依赖 echarts 和 echarts-for-react 两个包。

### 引入方式

```js
// app.js 全局注册
// echarts主题色配置
import { sui } from 'sui-components/lib/utils/echartTheme.js';
import echarts from 'echarts';
echarts.registerTheme('suiTheme', sui);

// xx.js内的组件上引入主题 theme='suiTheme'
import ReactEcharts from 'echarts-for-react';
<ReactEcharts theme="suiTheme" option={option} />;
```

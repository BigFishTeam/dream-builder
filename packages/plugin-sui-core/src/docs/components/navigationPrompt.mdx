## NavigationPrompt 页面跳转提示

``` js
import { NavigationPrompt } from 'sui-components';
```
### 何时使用

- 离开当前本页时，该页面有未保存的操作时，进行提醒
- 需要时，将该组件嵌入到当前页面的任何位置即可

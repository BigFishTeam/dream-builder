- 审核类业务，sui建议--
① 规律是左侧展示，右侧操作。
② 页面初始化收起导航。
③ 通过和不通过颜色提供了两种`sui-pass-bg-color`、`sui-delete-bg-color`。针对多种不通过的原因，不建议使用多种颜色，用`不通过`的颜色即可。
- 比较重要的按钮要加上二次确认提醒框。
- 关于按钮，sui建议--
① 页面最好是只有一个`主题色`的按钮，其他的使用次要的白色按钮，以区分主次，尤其是按钮集中的区域。
② sui提供`红色警告`按钮。

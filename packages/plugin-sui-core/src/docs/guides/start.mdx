## 快速开始


### 开始一个新项目

使用<a href="https://git.code.oa.com/comment_frontend/sigma-cli" target="_blank"  >sigma-cli</a>安装:


```
// 全局安装
tnpm install -g sigma-cli
// 初始化sui-seed
sigma init demo git@git.code.oa.com:comment_frontend/sui-seed.git
```

or 直接拷贝sui-seed地址使用:
``` 
git clone git@git.code.oa.com:comment_frontend/sui-seed.git
```

### 直接使用sui-components组件库

``` 
tnpm install sui-components
```

### 注意事项

- 本项目版本迭代较快，sui-seed内引用的 `sui-compoennts` 和  `sui-icons` 组件库请及时更新。
- 如果更新不生效，请先卸载，再安装。

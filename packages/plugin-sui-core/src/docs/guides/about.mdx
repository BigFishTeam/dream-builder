## 关于SUI

SUI是基于antd封装的一层业务组件库，适用于安全内容平台的中后台业务。

### 期望

- 提供统一、清爽的视觉方案
- 提供一致、顺畅的交互方案
- 提供通用业务模块的组件池
- 提高开发者的工作效率

### 设计理念

`Attractive Things Work Better` / 有吸引力的东西更好用

### 设计原则

- 内容一致性、可理解性
- 核心内容优先、降低视觉噪音
- 舒适感、掌控感
